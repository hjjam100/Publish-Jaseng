Triton.Span은 스팬을 표시할 때 사용합니다.
- Attribute
content 스팬의 내용을 표시
- Function
onClick 스팬을 클릭 시 발생하는 함수
- Theme
normal 기본 스타일의 스팬
CommentCount 댓글 숫자를 나타 내는 스팬
WriterAndRegisteredDate 작성자, 등록일시를 나타 내는 스팬
UnderlineWhenHover 마우스오버시 밑줄 그려지는 스팬
CommentWriter 댓글 작성자를 나타 내는 스팬
CommentRegisteredDater 댓글 등록일시를 나타 내는 스팬
- Sample Code
화면 상의 스팬을 표시한다.
