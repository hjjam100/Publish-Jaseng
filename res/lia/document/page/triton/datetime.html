Triton.DatetimePicker은 날짜를 입력 할 때 사용합니다.
- Function
onSelect 날짜 선택 시 발생 함수
setDisabled 날짜 선택 가능 여부
- Default Width
DATETIME_WIDTH Type이 DATETIME인 경우 넓이 지정 할 수 있다.
DATE_WIDTH Type이 DATE인 경우 넓이 지정 할 수 있다.
- Type
TYPE_DEFAULT 날짜와 시간을 입력
TYPE_DATE 날짜 입력
TYPE_DATETIME 날짜와 시간을 입력
- Theme
Normal 기본 스타일의 날짜 입력
- Sample Code
화면 상의 날짜 입력 뷰를 표시한다.
