Triton.KeyValueSection은 KEY와 VALUE를 한 줄에 표시할 때 사용합니다.
- Attribute
keyContent KEY 범위에 들어 갈 내용을 표시
valueContent VALUE 범위에 들어 갈 내용을 표시
keyOptions KEY 범위에 들어 갈 내용에 대한 옵션 정의
colonOptions KEY와 VALUE의 중간에 들어갈 콜론에 대한 옵션 정의
valueOptions VALUE 범위에 들어 갈 내용에 대한 옵션 정의
- Theme
Normal 기본 스타일의 테마
- Sample Code
화면 상의 KEY와 VALUE로 나누어져 표현하고 자 하는 범위를 표시한다.
