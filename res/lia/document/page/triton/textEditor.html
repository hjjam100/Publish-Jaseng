Triton.TextEditor은 텍스트에디터를 나타 낼 때 사용합니다.
- Attribute
textEditorOptions init한 텍스트에디터의 옵션을 지정 할 수 있다.
setValue 텍스트에디터의 값을 변경 할 수 있다.
getValue 텍스트에디터의 값을 가지고 올 수 있다.
- Function
initTextEditor 텍스트에디터를 초기화
setValue 텍스트에디터 값 변경
getValue 텍스트에디터 값 가지고오기
- Theme
Normal 기본 스타일의 텍스트에디터
Comment 댓글 스타일의 텍스트에디터
CommentList 댓글목록 스타일의 텍스트에디터
- Sample Code
화면 상의 텍스트에디터를 표시한다.
