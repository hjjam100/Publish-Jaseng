Triton.PhoneNumberInput은 전화번호를 나타 낼 때 사용합니다.
- Attribute
number1Options 전화번호의 시작 옵션
number2Options 전화번호의 두번째 옵션
number3Options 전화번호의 마지막 옵션
hyphenOptions 전화번호 연결 하이픈 옵션
- Attribute
getValue number1Option 값 + number2Option 값 + number3Option 값 가지고 올 수 있다.
setValue 들어온 데이터가 해당 뷰에 맞게 값 변경.
number3Options 전화번호의 마지막 옵션
hyphenOptions 전화번호 연결 하이픈 옵션
- DefaultValues
cellPhoneNumber1OptionList 핸드폰 번호의 첫번째 옵션리스트
homePhoneNumber1OptionList 유선 전화번호의 지역번호 옵션리스트
totalPhoneNumber1OptionList 핸드폰 + 유선 전화번호의 옵션리스트
officePhoneNumber1OptionList 핸드폰 + 유선 전화번호의 옵션리스트 ( Ex. 사내 번호 )
- Theme
Normal 기본 스타일의 전화번호 입력
- Sample Code
화면 상의 전화번호 입력 뷰를 표시한다.
