Triton.ListTable은 목록형 뷰를 나타 낼 때 사용합니다.
- Append Process
appendHeaderRow Header Row 배치
appendHeaderColumn Header Column을 Header Row에 배치
appendRow 해당 목록 데이터의 Row 배치
appendColumn 해당 목록 데이터를 배치한 Row에 배치
appendItem 최근 배치된 Column에 필요한 아이템 배치
- Row Theme
Normal 기본 스타일의 Row
Selected 선택된 스타일의 Row
Division 나누어진 스타일의 Row
NoLink 클릭 되지 않는 스타일의 Row
Header Header에서 쓰는 스타일의 Row
High 목록 상단에 위치하는 스타일의 Row
AlwaysOnTop 목록 상단에 항상 위치하는 스타일의 Row
- Column Theme
Normal 기본 스타일의 Column
Header Header에서 쓰이는 Column
- Sample Code
화면 상의 목록 테이블 뷰 범위를 표시한다.
