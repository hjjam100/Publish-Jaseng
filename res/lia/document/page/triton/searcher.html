Triton.Searcher은 검색 뷰 나타 낼 때 사용합니다.
- Attribute
searchOptionList 검색 시 필요한 Key, Value 리스트
searchSelectedValue 선택 된 Key 값
searchKeyword 검색 시 필요한 Value
- Function
onSearch 검색 시 함수
onSearchCallWhenInput 검색어 입력 시 검색 함수 활성화
setSearchKeyword 검색어 변경
setSearchSelectedValue 검색 Key 변경
getSearchKeyword 검색어 Get
getSearchSelectedValue 현재 검색 Key Get
- Theme
Normal 기본 스타일의 검색 뷰
Category 카테고리 스타일의 검색 뷰
- Sample Code
화면 상의 검색 뷰를 표시한다.
