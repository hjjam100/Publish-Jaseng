Triton.RadioButtonSection은 라디오버튼 표시할 범위를 나타 낼 때 사용합니다.
- Attribute
radioButtonTheme 라디오버튼의 테마를 지정 할 수 있다.
manual 작동 여부.
selectedValue 주어진 값이 있거나 선택 된 값이 필요할 경우 사용 할 수 있다.
optionList 필요한 갯수만큼 라디오버튼을 만들 수 있다.
optionMap 라디오버튼 Option을 Map 형식으로 만들 수 있다.
onSelected 라디오버튼 선택 시 작동하는 기능 작성 할 수 있다.
- Function
appendRadioButton 라디오버튼을 현재 범위 뒤에 append 할 수 있다.
setValue 라디오버튼의 값을 변경 할 수 있는 함수
getSelectedOption 현재 선택 된 라디오버튼의 옵션 정보를 가지고 올 수 있는 함수
getValue 현재 라디오버튼의 값을 가지고 올 수 있는 함수
setPressed 체크박스의 체크 여부를 변경 할 수 있는 함수
- Sample Code
화면 상의 라디오버튼 범위를 표시한다.
Triton.RadioButton은 라디오버튼 표시할 때 사용합니다.
- Attribute
content 라디오버튼 내용을 표시.
manual 작동 여부.
- Function
getOption 라디오버튼 옵션을 가지고 오는 함수
setStatus 라디오버튼의 상태를 변경 할 수 있는 함수
getStatus 체크박스의 현재 상태를 가지고 오는 함수
setValue 체크박스의 값을 변경 할 수 있는 함수
getValue 체크박스의 현재 값을 가지고 오는 함수
getPressed 체크박스의 현재 체크 여부를 가지고 오는 함수
setPressed 체크박스의 체크 여부를 변경 할 수 있는 함수
- Status
NORMAL 기본 상태의 라디오버튼
PRESSED 선택 된 상태의 라디오버튼
INACTIVE 비활성화 상태의 라디오버튼
- Theme
Line 라디오버튼이 라인에 하나씩 위치
Normal 기본 스타일의 라디오버튼
NoFirst 순서가 처음이 아닌 라디오버튼
- Sample Code
화면 상의 라디오버튼을 표시한다.
