Triton.FlagRadioButtonSection은 플래그 라디오버튼 표시할 범위를 나타 낼 때 사용합니다.
- Attribute
flagRadioButtonTheme 플래그 라디오버튼의 테마를 지정 할 수 있다.
onSelected 범위 안에 있는 플래그 라디오버튼을 선택 시 onSelected 내부에 작성한 코드대로 작동
optionList 필요한 갯수만큼 플래그 라디오버튼을 만들 수 있다.
- Function
appendFlagRadioButton 플래그 라디오버튼을 현재 범위 뒤에 append 할 수 있다.
setValue 플래그 라디오버튼의 값을 변경 할 수 있는 함수
getSelectedOption 현재 선택 된 플래그 라디오버튼의 옵션 정보를 가지고 올 수 있는 함수
getValue 현재 라디오버튼의 값을 가지고 올 수 있는 함수
- Theme
Normal 기본 스타일의 플래그 버튼섹션.
Category 카테고리 스타일의 플래그 버튼섹션.
Triton.FlagRadioButton은 플래그 형식의 라디오버튼을 나타 낼 때 사용합니다.
- Attribute
option 플래그 라디오버튼의 옵션을 지정.
manual 플래그 라디오버튼 조작 여부
link 플래그 라디오버튼 클릭시 이동 여부
theme 플래그 라디오버튼 테마
content 보여지는 플래그 라디오버튼 이름
pressed 플래그 라디오버튼 선택 여부.
onChecked 플래그 라디오버튼 선택 시 작동 하는 기능을 작성 할 수 있다..
- Function
appendRadioButton 라디오버튼을 현재 범위 뒤에 append 할 수 있다.
onClickMoreButton 더보기 버튼을 눌렀을 때 기능
- Status
NORMAL 플래그 라디오버튼 기본 상태
PRESSED 플래그 라디오버튼이 눌려진 상태
INACTIVE 플래그 라디오버튼이 동작 하지 않는 상태
- Theme
Normal 기본 스타일의 플래그 라디오버튼
Category 카테고리 스타일의 플래그 라디오버튼
CategoryNext 카테고리 플래그 라디오버튼이 동작 하지 않는 상태
- Sample Code
화면 상의 플래그 라디오버튼을 표시한다.
