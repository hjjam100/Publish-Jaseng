Triton.FlagSpan은 플래그를 표시할 때 사용합니다.
- Attribute
content 플래그의 내용을 표시
spanOptions Theme의 Attibute에서 span의 관련된 사항 조작
imgOptions Theme의 Attibute에서 img의 관련된 사항 조작
onClick 클릭시 지정 된 Theme에 따라 순서가 변경되거나 이름이 삭제 된다.
- Function
onClick 스팬을 클릭 시 발생하는 함수
- Theme
OrderBy 해당 컬럼에 속해 있는 데이터들의 순서를 변경 할 수 있는 플래그
NamDelete 해당 컬럼에 속해 있는 이름을 삭제 할 수 있는 플래그
- Sample Code
화면 상의 플래그를 표시한다.
