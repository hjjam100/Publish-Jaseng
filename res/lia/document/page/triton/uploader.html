Triton.Uploader은 파일을 업로드 할 때 사용합니다.
- Attribute
fileFormName 파일 업로드 시 Form 이름.
parameterMap 파일 업로드시 필요한 파라미터 지정.
maxUploadCount 파일 업로드 최대 갯수.
isMovieUploader 동영상 업로드 가능 여부.
- Function
setAttachmentList 업로드 된 파일 세팅
getAttachmentList 업로드 된 파일 값 가져오기
add 업로드 된 파일 Add
addAll 업로드 된 파일 전부 Add
setValue 업로드 된 파일 값 변경
getValue 업로드 된 파일 값 가져오기
- Sample Code
화면 상의 파일 업로드 뷰를 표시한다.
