Triton.Section은 작업 할 페이지 범위를 표시할 때 사용합니다.
- Attribute
content 범위의 내용을 표시
- Theme
Normal 기본 스타일의 범위
UnderlineWhenHover 마우스 오버 시 밑줄 생기는 스타일의 범위
Hover 마우스 오버 스타일의 범위
Square 사각형 스타일의 범위
ListMessage 목록 테이블에 메시지 넣는 스타일의 범위
CategoryLayout 카테고리를 넣을 수 있는 범위
LeftRightLayout 카테고리를 넣을 수 있는 범위
ButtonLayout 버튼을 넣을 수 있는 범위
LeftButtonLayout 버튼을 왼쪽으로 배치 해 넣을 수 있는 범위
RightButtonLayout 버튼을 오른쪽으로 배치 해 넣을 수 있는 범위
Pager 페이징 스타일의 범위
Category 카테고리 스타일의 범위
CommentButtonLayout 댓글 버튼 레이아웃 스타일의 범위
CommentButtonLayout2 오른쪽에 위치한 댓글 버튼 레이아웃 스타일의 범위
CommentWrite 댓글 작성자를 나타내는 범위
CommentWrite2 댓글 작성자를 나타내는 범위 ( 아래 margin이 존재 하지 않음. )
CommentItem 댓글을 보여주는 범위
CommentItem2 댓글을 보여주는 범위 ( 배경색 존재, 위 border 점선으로 표시 )
CommentHeader 댓글 Header를 나타내는 범위 ( 아래 margin 존재. )
CommentHeader2 댓글 Header를 나타내는 범위
