Triton.EmailInput은 이메일 입력 뷰를 나타 낼 때 사용합니다.
- Attribute
nameOptions 이메일 아이디 입력 뷰 옵션
domainOptions 이메일 도메인 주소 입력 뷰 옵션
directOptions 이메일 도메인 주소 직접 입력 뷰 옵션
atOptions 골뱅이 기호 옵션
- Function
getValue nameInput 값과 directInput 값 가지고 올 수 있다.
setValue 들어온 데이터로 해당 뷰에 맞게 값 변경 할 수 있다.
directOptions 이메일 도메인 주소 직접 입력 뷰 옵션
atOptions 골뱅이 기호 옵션
- DefaultValues
emailOptionList 이메일 도메인 주소리스트
militaryEmailOptionList MMOOC 이메일 도메인 주소리스트
- Theme
Normal 기본 스타일의 이메일 입력
- Function
setEmail 이메일 값을 변경 할 수 있는 함수
- Sample Code
화면 상의 이메일 입력 뷰를 표시한다.
