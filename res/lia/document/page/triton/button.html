Triton.FlatButton은 버튼을 표시할 때 사용합니다.
- Attribute
defaultColor 디폴트 버튼 색
hoverColor 마우스 오버 했을 때 버튼 색
content 버튼 내용을 표시
onClick 클릭하면 onClick에 작성한 내용대로 작동
- Theme
Square 네모 모양의 버튼
Header 헤더에 위치하는 버튼
HeaderShade 헤더에 위치하는 회색 버튼
Normal 기본 버튼
NormalLeft 왼쪽에 위치 기본 버튼
NormalRight 오른쪽에 위치 기본 버튼
NoFlat 플랫 없는 버튼
Delete 삭제 버튼
DeleteLeft 왼쪽에 위치 삭제 버튼
DeleteRight 오른쪽에 위치 삭제 버튼
DetailDelete 상세페이지에서 쓰는 삭제 버튼
Shade 회색 버튼
ShadeSelected 클릭 된 모양의 회색 버튼
Search 검색 버튼
ListNormalFlat 목록에서 쓰는 일반 플랫 버튼
ListSearch 목록에서 쓰는 검색 버튼
ShadeLeft 왼쪽에 위치 회색 버튼
ShadeRight 오른쪽에 위치 회색 버튼
ExcelLeft 왼쪽에 위치 엑셀 다운로드 버튼
Excel 엑셀 다운로드 버튼
Refresh 새로고침 버튼
ListEdit 목록에서 사용하는 편집 버튼
BigListEdit 목록에서 사용하는 큰 편집 버튼
ListInquiry 목록에서 사용하는 조회 버튼
ListDelete 목록에서 사용하는 삭제 버튼
ListDeleteLeft 왼쪽에 위치 목록에서 사용하는 삭제 버튼
ListDeleteRight 오른쪽에 위치 목록에서 사용하는 삭제 버튼
ListEditLeft 왼쪽에 위치한 목록에서 사용하는 편집 버튼
ListEditRight 오른쪽에 위치한 목록에서 사용하는 편집 버튼
ListInquiryLeft 왼쪽에 위치한 목록에서 사용하는 조회 버튼
ListInquiryRight 오른쪽에 위치한 목록에서 사용하는 조회 버튼
ListNormal 목록 범위에서 기본적으로 사용하는 버튼
DetailNormal 상세 페이지 범위에서 기본적으로 사용하는 버튼
ListNoFlat 목록 범위에서 사용하는 플랫 없는 모양의 버튼
- Sample Code
화면 상의 버튼를 표시한다.
