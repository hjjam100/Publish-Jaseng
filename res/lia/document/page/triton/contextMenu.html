Triton.ContextMenu는 컨텍스트메뉴를 나타 낼 때 사용합니다.
- Attribute
hideWhenClick 클릭 시 숨김
removeWhenClick 클릭 시 삭제
onItemClickList 메뉴 아이템 클릭 시 리스트
onItemClick 메뉴 아이템 클릭
- Function
setPosition 컨텍스트 메뉴 위치 변경
setWidth 컨텍스트 메뉴 넓이 변경
add 컨텍스트 메뉴 추가
- ContextMenu Theme
Text 컨텍스트 메뉴 TEXT 스타일의 메뉴
Button 컨텍스트 메뉴 버튼 스타일의 메뉴
- ContextMenuItem Theme
Text 컨텍스트 메뉴 아이템의 TEXT 스타일 메뉴
TextNoPadding 컨텍스트 메뉴 아이템의 패딩 없는 TEXT 스타일 메뉴
Button 컨텍스트 메뉴 아이템의 버튼 스타일 메뉴
TextNoPadding 컨텍스트 메뉴 아이템의 패딩 없는 TEXT 스타일 메뉴
- Sample Code
화면 상의 컨텍스트 메뉴를 표시한다.
