Triton.SubTab은 서브탭을 표시할 때 사용합니다.
- Function
onBind 서브탭 버튼 클릭시 함수
onAdd 서브탭 버튼 추가시 함수
addSubTabButton 서브탭 버튼 추가
getSubTabIndex 현재 선택 되어진 서브탭 인덱스
setSubTabIndex 서브탭 인덱스 변경
getSubTabButtonOptionsList 현재 서브탭 옵션리스트
getSelectedSubTabButtonOptions 현재 선택 되어진 서브탭 옵션리스트
- Theme
normal 기본 스타일의 서브탭
- Sample Code
화면 상의 서브탭을 표시한다.
