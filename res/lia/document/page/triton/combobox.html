Triton.ComboBox은 데이터를 선택 할 때 사용합니다.
- Attribute
selectedValue 선택 박스에서 선택되어 보여지는 값.
maxSelectCount 모드가 SELECT에서 POPUP으로 변경되는 조건이 되는 Count.
mode 선택 박스의 모드를 지정 할 수 있다.
optionList 선택 박스에 들어가는 Option 목록
optionMap 선택 박스에 들어가는 맵 형식의 Option
- Function
setTheme 선택 박스의 Theme를 변경
setDisabled 선택 박스의 Disabeld를 변경
setOptionList 선택 박스의 optionList를 변경
setValue 선택 박스의 값을 변경
getValue 선택 박스의 선택 되어진 값
getSelectedOption 선택 박스의 선택 되어진 값의 옵션 값
onSelected 선택 박스를 선택했을때 함수
- Theme
Normal 기본 스타일의 선택박스
Full Full 스타일의 선택박스
SubMenuTitle 서브메뉴에서 쓰이는 스타일의 선택박스
SubMenuTitleSelectWhite 서브메뉴에서 쓰이는 흰색으로 선택되는 선택박스
Category 카테고리 스타일의 선택박스
CategoryNotFirst 첫 카테고리 다음에 쓰이는 스타일의 선택박스
- Mode
SELECT 기본 모드의 선택 박스
POPUP 팝업 모드의 선택 박스
MIX 데이터의 갯수에 따른 모드 ( 갯수가 많을 경우 = POPUP / 갯수가 많지 않을 경우 = SELECT )
- Sample Code
화면 상의 선택박스 뷰를 표시한다.
