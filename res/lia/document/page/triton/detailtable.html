Triton.DetailTable은 상세 정보 뷰를 나타 낼 때 사용합니다.
- Append Process
appendRow 해당 테이블 Row
appendKeyColumn 해당 테이블에 배치한 Row에 Key형태의 Column 배치
appendValueColumn 해당 테이블에 배치한 Row에 Value형태의 Column 배치
appendItem 해당 테이블에 배치한 Column에 Item 추가
- Attribute
keyColumnWidth 배치한 keyColumn의 Width를 지정
- Sample Code
화면 상의 상세 테이블 범위를 표시한다.
