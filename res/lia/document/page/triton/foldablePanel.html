Triton.FoldablePanel은 접혀 있는 테이블 표시할 때 사용합니다.
- Attribute
title 접혀 있는 뷰를 클릭 시 펼칠 수 있는 텍스트
contentMinWidth 접혀 있는 판넬에 붙어있는 콘텐츠의 최소 넓이 지정
contentMinHeight 접혀 있는 판넬에 붙어있는 콘텐츠의 최소 높이 지정
folded 접힘 여부
- Function
fold 접힘 여부를 결정
- Theme
normal 기본 스타일의 접혀 있는 판넬
Padding 위에 margin이 있는 접혀 있는 판넬
- Sample Code
화면 상의 접혀 있는 판넬을 표시한다.
