Triton.AttachmentItem은 첨부파일 표시할 때 사용합니다.
- Attribute
fileName 첨부파일 이름 표시
fileSize 첨부파일 크기 표시
fileUrl 첨부파일 URL 표시
imgOptions 첨부파일을 나타내는 기본 이미지에 대한 옵션 정의
nameOptions 첨부파일 명에 대한 옵션 정의
sizeOptions 첨부파일 크기에 대한 옵션 정의
- Theme
Normal 기본 스타일의 첨부파일 정보
SIZE_NONE 첨부파일 크기가 나오지 않는 첨부파일 정보
- Sample Code
화면 상의 파일 정보 뷰를 표시한다.
