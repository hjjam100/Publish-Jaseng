Triton.DatetimePeriodPicker은 날짜와 시간을 입력 할 때 사용합니다.
- Option
startOptions 시작일에 대한 옵션 정의
endOptions 종료일에 대한 옵션 정의
- Function
getStartDatetimePicker 시작일에 대한 DatePicker에 정보
getEndDatetimePicker 종료일에 대한 DatePicker에 정보
- Type
TYPE_DEFAULT 날짜와 시간을 입력
TYPE_DATE 날짜 입력
TYPE_DATETIME 날짜와 시간을 입력
- Sample Code
화면 상의 날짜와 시간 입력 뷰를 표시한다.
