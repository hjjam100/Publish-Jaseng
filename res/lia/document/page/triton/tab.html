Triton.Tab은 탭을 표시할 때 사용합니다.
- Attribute
addButtonHidden 탭 추가 버튼 숨기기
listButtonHidden 탭 목록 보기 버튼 숨기기
leftButtonHidden 탭 왼쪽으로 이동 버튼 숨기기
rightButtonHidden 탭 오른쪽으로 이동 버튼 숨기기
- Function
onBind 탭 버튼 클릭시 함수
onAdd 탭 버튼 추가시 함수
getTabName 현재 선택 되어진 탭 명
setTabName 현재 선택 되어진 탭 명 변경
addTabButton 탭 버튼 추가
getTabIndex 현재 선택 되어진 탭 인덱스
setTabIndex 탭 인덱스 변경
getTabButtonOptionsList 현재 탭 옵션리스트
- Theme
normal 기본 스타일의 탭
- Sample Code
화면 상의 탭을 표시한다.
