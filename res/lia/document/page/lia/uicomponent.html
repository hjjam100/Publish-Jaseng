· Introduction
Lia에서는 jQuery 플러그인 형식으로 구현된 UI Component를 제공합니다.
DOM객체의 class와 attribute를 이용하여 상태 및 정보를 저장합니다.
제공되는 UI Component는 다음과 같습니다.
button 마우스 이벤트를 이용한 버튼 컴포턴트
image_button 마우스 이벤트에 따른 이미지 버튼 컴포넌트
image_cover 표시될 영역에 이미지 비율 깨지지 않도록 표시 컴포넌트
loading_indicator 로딩 버튼 컴포넌트
popup 팝업 컴포넌트
popup.loading 팝업+로딩 버튼 컴포넌트

· 초기화

Lia에서 제공하는 UI Component를 사용하려면 초기화를 하여야 합니다.
초기화 함수눈 init + 컴포넌트 명으로 네이밍 되었습니다.
초기화가 완료되면 inited class가 dom에 설정되며 초기화는 한번만 진행되며 두번째 이후 실행되면 무시합니다.
예시는 다음과 같습니다.
· button
button은 마우스 이벤트에 대한 상태 ( hovering, pressing, pressed ) class로 설정해 주며 Lia.Component.Event.STATUS_CHANGED 이벤트가 상태가 변경될 때 실행됩니다.
다른 button 류 Component들도 동일하게 동작합니다.
- 기본형
<div class="button"></div>
- Event Flag Class
default 기본 상태
hovering 마우스가 해당 객체에 올라갔을 때
pressing 마우스 버튼이 눌렸을 때
pressed 버튼이 pressed 상태로 체크되었을 때 ( .buttonPressed 함수로 설정 )
- Functional Flag Class
no_bind 초기화 될 때 마우스 이벤트 관련 이벤트에 매핑하지 않음
no_change button 관련 이벤트가 발생했을 때 관련 이벤트 처리를 하지 않음
- Event Flag Class
hovering 마우스가 해당 객체에 올라갔을 때
pressing 마우스 버튼이 눌렸을 때
pressed 버튼이 pressed 상태로 체크되었을 때 ( .buttonPressed 함수로 설정 )
- Event
.on(Lia.Component.Event.STATUS_CHANGED, function(e) {} ) 버튼 상태값이 변경되면 실행됩니다.
e.status에 상태값이 기록되어서 내려옵니다.
Lia.Component.Value.Button.Status.DEFAULT 기본 상태
Lia.Component.Value.Button.Status.HOVERING 마우스 올라간 상태
Lia.Component.Value.Button.Status.PRESSING 마우스 누른 상태
Lia.Component.Value.Button.Status.PRESSED 마우스 눌러진 상태
- Function
.initButton() 초기화
.buttonPressed() 현재 pressed 상태인지 아닌지 리턴
.buttonPressed( pressed ) pressed 상태를 설정
- Sample Code
화면상에 button 객체를 초기화하고 css를 각 상태에 맞는 css를 설정하여 표시한다.
button의 상태 변경 status event를 받아 텍스트를 버튼에 설정한다.
클릭햇을 때 pressed 상태를 설정하거나 해제한다.
· image_button
image_button은 button의 기능올 함께 사용하는 UI Component로 button의 기능은 모두 사용 가능합니다.
button과 마찬가지로 마우스 이벤트에 대한 상태 ( hovering, pressing, pressed ) class로 설정해 주며 해당 상태에 따른 이미지를 표시하게 해줍니다.
또한 button과 동일하게 Lia.Component.Event.STATUS_CHANGED 이벤트가 상태가 변경될 때 실행됩니다.

lia-src로 받은 이미지 url ( 없으면 src를 사용 )를 가지고 해당 상태에 맞는 이미지를 보여줍니다.
이미지 주소의 변경 형식은 정해져 있으며, _hovering.png, _pressed.png를 접미사로 사용하여 파일들을 보여주게 됩니다.

DOM에 설정되어 있는 class가 image_button 만 존재한다면 pressing, pressed 상태일 때 _pressed.png 파일을 표시합니다.
DOM에 설정되어 있는 class가 image_button hover 이면 hovering 상태일때 _hovering.png 이미지를 보여주고 pressing, pressed 상태일 때 _pressed.png 파일을 표시합니다.
DOM에 설정되어 있는 class가 image_button press 이면 hovering, pressing, pressed 일때 _pressed.png 이미지를 보여주게 됩니다.
- 기본형
<img class="image_button" lia-src="표시할 이미지 경로"/>
- Attribute
lia-src 표시할 이미지 경로, 없으면 src 속성을 가지고와서 사용함
- Functional Flag Class
no_binding 초기화 될 때 마우스 이벤트 관련 이벤트에 매핑하지 않음
no_change button 관련 이벤트가 발생했을 때 관련 이벤트 처리를 하지 않음
hover hovering 상태일때 _hovering.png 이미지를 보여주고 pressing, pressed 상태일 때 _pressed.png 파일을 표시
press hovering, pressing, pressed 일때 _pressed.png 이미지를 표시
- Event Flag Class
hovering 마우스가 해당 객체에 올라갔을 때
pressing 마우스 버튼이 눌렸을 때
pressed 버튼이 pressed 상태로 체크되었을 때 ( .buttonPressed 함수로 설정 )
- Event
.on(Lia.Component.Event.STATUS_CHANGED, function(e) {} ) 버튼 상태값이 변경되면 실행됩니다.
e.status에 상태값이 기록되어서 내려옵니다.
Lia.Component.Value.Button.Status.DEFAULT 기본 상태
Lia.Component.Value.Button.Status.HOVERING 마우스 올라간 상태
Lia.Component.Value.Button.Status.PRESSING 마우스 누른 상태
Lia.Component.Value.Button.Status.PRESSED 마우스 눌러진 상태
- Function
.initImageButton() 초기화
.imageButtonPressed() 현재 pressed 상태인지 아닌지 리턴
.imageButtonPressed( pressed ) pressed 상태를 설정
.setImageButtonByStatus( status ) status를 설정
- Sample Code
image_button 기본형, image_button press, image_button hover 이 어떻게 동작하는지 순서대로 보여줍니다.
· image_cover
표시핧 영역에 맞추어 이미지 비율이 맞도록 긴 부분을 보여지지 않게 하여 화면상에 표시합니다.
- 기본형
<div class="image_cover" lia-src="표시할 이미지 경로"></div>
- Attribute
lia-src 표시할 이미지 경로
lia-error-src 이미지 로딩이 에러났을 때 표시할 이미지
lia-width 이미지 원보 사이즈 width, 입력되어 있지 않으면 img 태그의 load 이벤트에서 기록됨
입력 하면 이미지 로딩 속도가 느릴 때 이미지 크기 조정을 화면상에서 보이지 않게 할 수 있음
lia-height 이미지 원보 사이즈 height, 입력되어 있지 않으면 img 태그의 load 이벤트에서 기록됨
입력 하면 이미지 로딩 속도가 느릴 때 이미지 크기 조정을 화면상에서 보이지 않게 할 수 있음
- Event Flag Class
image_error 이미지 로딩 에러일 때
- Function
.initImageCover() 초기화
.updateImageCover 현재 사이즈에 맞추어 이미지 표시 업데이트
- Sample Code
원본 이미지와 image_cover 적용된 화면을 보여주며 창을 줄였다 키우면 화면 사이즈에 따라 어떤 식으로 이미지가 보여지는 지 확인 할 수 있습니다.
· loading_indicator
로딩 이미지를 순차적으로 표시합니다.
lia-src 속성의 url 형식을 가지고 lia-start-index 부터 lia-end-index 까지의 이미지를 순차적으로 보여주게 됩니다.
- 기본형
<img class="loading_indicator" lia-src="{img 주소 format:/loading/{index}.png}" lia-index="{현재 인덱스}" lia-start-index="{시작 인덱스}" lia-end-index="{끝인덱스}" alt="loading indicator"/>
- Attribute
lia-src 표시할 URL 형식, {index} 라고 표시된 부분을 표시할 인덱스 숫자로 처리한다.
lia-index 현재 표시할 이미지
lia-start-index 로딩을 표시할 시작 이미지 인덱스
lia-end-index 로딩을 표시할 마지막 이미지 인덱스
- Function
.initLoadingIndicator() 초기화
.playLoadingIndicator() 로딩 이미지 표시 시작
.pauseLoadingIndicator() 로딩 이미지 표시 중지
.initAndPlayLoadingIndicator 초기화 및 로딩 이미지 표시 시작
- Sample Code
img_1.png ~ img_8.png 까지 준비되어 있는 이미지로 로딩 이미지 표시 시작
· popup
popup를 div popup을 페이지에 표시하기 위한 인터페이스를 제공합니다.
dim을 활용하여 popup의 뒷 배경을 표시하고 그 위에 popup를 표시합니다.
dim 과 popup 양측에 서로 연결되는 popup과 dim을 설정해 주어야 합니다.
- 기본형
<div class="dim [cancelable]" lia-z-index="{z-index 값}" lia-popup="{연결된 popup Selector}" lia-opacity="{투명도 값}" lia-dim-background-color="{색깔}"></div> <div class="popup" style="width:{popup width};height:{popup height}" lia-z-index="{z-index 값}" lia-position-top="{위치 값 고정 시}" lia-position="{위치 : fiexed, absolute}" lia-dim="{사용할 dim Selector}"></div>
