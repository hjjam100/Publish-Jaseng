· 사용을 위한 준비
Lia를 사용하려면 lia에 관련된 css 파일과 js 파일들을 참조하여야 합니다.
jQuery 기반 위에서 구현되었기 때문에 jQuery 및 필요한 기능들에 대한 라이브러리 등을 참조하여야 합니다.

· 경로

Lia의 파일들은 웹 페이지의 /res/lia 경로에 존재하는 것을 가정합니다.
만약 해당 경로 외의 경로로 설정하시려면 리소스 파일들의 경로들을 모두 변경하여야 합니다.
